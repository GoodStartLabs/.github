## Why games?

Games create environments with:

- **Clear objectives + measurable outcomes**
- **Natural human feedback loops** (votes, preferences, spectatorship)
- **Adversarial & multi-agent dynamics** that stress-test models beyond single-turn tasks

This makes games a practical, scalable route to evaluating and improving alignment in-the-wild.
